# FacilitisAPI
Api for simple minecraft plugin creation

Add this to your pom.xml:
```
<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>
```
```
<dependency>
  <groupId>com.github.MrLiam2614</groupId>
  <artifactId>FacilitisAPI</artifactId>
  <version>v1.0.0</version>
</dependency>
```
